# Student Management System (Java)

## Project Overview
The **Student Management System** is a Java console application to manage student records efficiently. Users can add, view, update, search, and remove student information through a menu-driven interface. Each student has a unique five-digit ID automatically generated by the system.  

This project demonstrates my abillity in **object-oriented programming (OOP)** principles, input validation, error handling, and the use of Java collections.

---

## Features
- Add a new student (name, surname, age, auto-generated ID)
- View all students sorted by ID
- Update existing student details
- Search students by name, surname, or ID
- Remove students by name
- Input validation with error alerts (JOptionPane + beep using the Toolkit)
- Menu-driven interface for easy navigation

---

## Requirements
- **Java Development Kit (JDK) 21 or higher** installed
- A code editor or IDE such as:
  - IntelliJ IDEA
  - Eclipse
  - VS Code with Java extensions
- Basic knowledge of running Java programs from the console
